the project has three folders and two files :

	- src : contains the different classes of the project
	- Tests : contains the test classes
	- javaDoc : the documentation of the project 
	- config.txt : an example file that contains the configuration of the simulation 
	- data.txt : an examle of the input data of the simulation 

to read the documentation of the project :
	- open the javaDoc folder 
	- then open the index.html  file
